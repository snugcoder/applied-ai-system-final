# PawPal+ - AI Final

## Getting started

### Setup

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Run the script by typing ```python main.py``` in the terminal.
